#Sayı İşlemleri Programı
Bu proje kullanıcının girdiği sayılar üzerinden:
- Ortalama
- Mod
- Medyan

hesaplayan basit bir Python uygulamasıdır.

---

#Kullanım
Program çalıştığında kullanıcıdan sayılar istenir.
- Sayıları tek tek girin.
- İşlemleri bitirmek için **0** tuşlayın.
- Program girilen listeyi ve sonuçları ekranda gösterir.

---

#Çalıştırma
Bilgisayarınızda **Python 3.x** kurulu olmalıdır.
Terminalde (CMD / PowerShell / VS Code terminal) şu komutu çalıştırın:

```bash
python islem.py


Özellikler:
Kullanıcıdan sınırsız sayı alabilir.
Boş liste kontrolü yapılır.
Sıralama işlemi ile doğru medyan bulunur.
statistics modülü ile mod güvenilir şekilde hesaplanır.

Kullanılan Python Konuları:
while True döngüsü,Koşul yapıları,Fonksiyon yazımı,Liste işlemleri,statistics.mode,Sıralama ve indeksleme
